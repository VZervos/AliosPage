# Media Files Directory

Place your images and videos in the following folders:

## Folder Structure

### `images/logos/`
- **main-logo.png** - Main logo for hero section (homepage, about page)
- **footer-logo.jpg** - Small logo for footer (all pages)

### `images/photos/`
- **performance-01.jpg** through **performance-08.jpg** - Performance photos for homepage carousel
- **collaboration-01.jpg** through **collaboration-03.jpg** - Collaboration page photos

### `images/training/`
- **training-01.jpg** - Training photo 1 (homepage, join page)
- **training-02.jpg** - Training photo 2 (homepage, join page)
- **training-03.jpg** - Training photo 3 (join page)

### `videos/`
- **training-video-thumb.jpg** - Thumbnail for training video (join page)
- **training-video.mp4** - Training video file (TODO: add when ready)
- **performance-video-thumb.jpg** - Thumbnail for performance video (collaborate page)
- **performance-video.mp4** - Performance video file (TODO: add when ready)

## Notes
- All external URLs have been commented out in the HTML files
- Replace the placeholder file names with your actual media files
- Keep the same file names or update the HTML references accordingly
- Recommended image formats: JPG, PNG, WebP
